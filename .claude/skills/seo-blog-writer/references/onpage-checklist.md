## Pre-Writing Research

### Keyword Analysis
- [ ] Primary keyword identified
- [ ] Search volume verified (100+ monthly searches)
- [ ] Keyword difficulty assessed (target under 30-50)
- [ ] Search intent identified (informational/transactional/etc.)
- [ ] SERP features analyzed (featured snippets, PAA, etc.)

### Competitor Analysis
- [ ] Top 5-10 ranking pages reviewed
- [ ] Average word count noted
- [ ] Content format/structure analyzed
- [ ] Gaps and opportunities identified
- [ ] Unique angle determined

### Secondary Keywords
- [ ] 3-5 secondary keywords identified
- [ ] LSI/semantic keywords listed
- [ ] Question-based keywords noted (for FAQ)
- [ ] Long-tail variations documented

## Title Tag Optimization

### Requirements
- [ ] Primary keyword included (preferably at beginning)
- [ ] Under 60 characters (55-60 ideal)
- [ ] Unique to this page (no duplicates on site)
- [ ] Compelling and click-worthy
- [ ] Accurately describes content
- [ ] No keyword stuffing

### Title Tag Formulas

**How-to posts:**
- "How to [Achieve X]: [Number] [Adjective] [Strategies/Tips/Steps]"
- Example: "How to Write SEO Content: 9 Proven Strategies"

**List posts:**
- "[Number] Best [Topic] [for X] in [Year]"
- Example: "15 Best SEO Tools for Beginners in 2025"

**Guide posts:**
- "[Topic]: The Complete [Guide/Checklist] for [Year]"
- Example: "Keyword Research: The Complete Guide for 2025"

**Comparison posts:**
- "[Option A] vs [Option B]: Which [Benefit] Better?"
- Example: "Ahrefs vs SEMrush: Which Finds Keywords Better?"

### Power Words for Titles
- Ultimate, Complete, Definitive, Essential
- Proven, Effective, Powerful, Advanced
- Step-by-Step, Easy, Quick, Simple
- [Current Year] - adds freshness
- Numbers (odd numbers often perform better)

## Meta Description

### Requirements
- [ ] Under 155-160 characters
- [ ] Primary keyword included naturally
- [ ] Compelling value proposition
- [ ] Clear call-to-action
- [ ] Matches content accurately
- [ ] Unique to this page

### Meta Description Formula
```
[What the content covers] + [Key benefit to reader] + [CTA]
```

**Example:**
> "Learn the 12 essential on-page SEO techniques that top-ranking pages use. Includes actionable checklist and real examples. Start optimizing today."

### CTA Examples
- "Learn how" / "Discover"
- "Get started" / "Start [doing X]"
- "Read the guide" / "See examples"
- "Find out" / "Explore"

## URL Structure

### Requirements
- [ ] Primary keyword included
- [ ] Short (3-5 words ideal)
- [ ] Hyphens between words
- [ ] Lowercase letters only
- [ ] No special characters
- [ ] No dates/years (unless time-sensitive)
- [ ] No stop words (a, the, and, of, etc.)

### URL Examples

**Good:**
- `/seo-content-writing-guide/`
- `/best-keyword-research-tools/`
- `/on-page-seo-checklist/`

**Bad:**
- `/how-to-write-the-best-seo-content-for-your-website-in-2025/`
- `/post_id=12345/`
- `/blog/article/seo/tips/content/`

## Header Tags

### H1 Tag
- [ ] Only one H1 per page
- [ ] Primary keyword included
- [ ] Matches title tag intent (can differ slightly)
- [ ] Clear topic indication

### H2 Tags
- [ ] Used for main sections
- [ ] Include secondary keywords naturally
- [ ] Describe section content accurately
- [ ] 3-8 H2s typical for 1500-2500 word posts

### H3+ Tags
- [ ] Used for subsections under H2s
- [ ] Maintain logical hierarchy
- [ ] Include long-tail keywords where natural
- [ ] Aid scanability

### Header Keyword Placement

```markdown
# [H1: Primary Keyword + Hook]

## [H2: Secondary Keyword 1 - Main Section]

### [H3: Long-tail variation or subtopic]

### [H3: Another subtopic]

## [H2: Secondary Keyword 2 - Next Section]

## [H2: Related Topic/FAQ Section]
```

## Content Optimization

### Keyword Placement
- [ ] Primary keyword in first 100 words
- [ ] Primary keyword in at least one H2
- [ ] Natural keyword density (1-2%, don't force)
- [ ] Semantic keywords throughout
- [ ] Keyword in conclusion

### Content Quality
- [ ] Comprehensive topic coverage
- [ ] Answers main query completely
- [ ] Unique insights/value added
- [ ] Accurate, fact-checked information
- [ ] Up-to-date data and examples
- [ ] No fluff or filler content

### Readability
- [ ] Short paragraphs (2-4 sentences)
- [ ] Varied sentence length
- [ ] Bullet points and lists used
- [ ] White space for visual breaks
- [ ] Reading level appropriate for audience
- [ ] Transition words for flow

### Engagement Elements
- [ ] Hook in introduction
- [ ] Clear value proposition upfront
- [ ] Examples and case studies
- [ ] Data and statistics
- [ ] Images, charts, or diagrams
- [ ] FAQ section (if relevant)
- [ ] Clear call-to-action

## Internal Linking

### Requirements
- [ ] 3-5 internal links per 1,000 words minimum
- [ ] Links to relevant, related content
- [ ] Descriptive anchor text (not "click here")
- [ ] Links distributed throughout content
- [ ] Links to pillar/cornerstone content
- [ ] No broken internal links

### Anchor Text Guidelines

**Good anchor text:**
- "our complete guide to keyword research"
- "learn more about on-page SEO factors"
- "these email marketing best practices"

**Bad anchor text:**
- "click here"
- "read more"
- "this article"
- Exact match keyword every time (over-optimization)

---

## External Linking

### Requirements
- [ ] 2-5 external links to authoritative sources
- [ ] Links support claims and data
- [ ] No links to competitors (usually)
- [ ] Links open in new tab
- [ ] No broken external links
- [ ] Avoid link to low-quality sites

### Best Sources to Link To
- Government sites (.gov)
- Educational institutions (.edu)
- Major news outlets
- Industry research/studies
- Official documentation
- Recognized experts' sites

## Image Optimization

### Every Image Needs
- [ ] Descriptive filename (keyword-rich)
- [ ] Alt text with keyword (naturally written)
- [ ] Compressed for fast loading
- [ ] Appropriate dimensions
- [ ] Relevant to content

### File Naming
**Good:** `on-page-seo-checklist-infographic.png`
**Bad:** `IMG_20240315_screenshot.png`

### Alt Text Examples
**Good:** "Checklist showing 10 essential on-page SEO factors"
**Bad:** "image" or "SEO SEO optimization SEO tips"

## Schema Markup

### Recommended Schema Types

**Article Schema (for blog posts):**
- headline
- author
- datePublished
- dateModified
- image
- publisher

**FAQ Schema (if FAQ section included):**
- Increases SERP real estate
- Can trigger rich results

**HowTo Schema (for tutorials):**
- Steps with descriptions
- Images for each step
- Time required

### Implementation
- Use JSON-LD format (Google preferred)
- Validate with Google Rich Results Test
- Don't mark up content not on page

## Technical Elements

### Page Speed
- [ ] Target under 3 seconds load time
- [ ] Images compressed and optimized
- [ ] Minimal render-blocking resources
- [ ] Browser caching enabled
- [ ] CDN used (if applicable)

### Mobile Optimization
- [ ] Mobile-responsive design
- [ ] Text readable without zooming
- [ ] Buttons/links easily tappable
- [ ] No horizontal scrolling
- [ ] Core Web Vitals passing

### Indexability
- [ ] Page not blocked by robots.txt
- [ ] No "noindex" tag (unless intentional)
- [ ] Canonical URL set correctly
- [ ] In XML sitemap

## Final Quality Check

### Before Publishing
- [ ] Spell check and grammar check complete
- [ ] All links tested and working
- [ ] Images loading correctly
- [ ] Mobile preview reviewed
- [ ] Page speed tested
- [ ] Schema validated
- [ ] No duplicate content issues
- [ ] Social sharing preview checked

### After Publishing
- [ ] Request indexing in Google Search Console
- [ ] Share on social media
- [ ] Update internal links on older posts
- [ ] Monitor for any crawl errors
- [ ] Track rankings for target keywords
